    scn_Bootstrapper

    This scene serves as a launching point for the application and is used as a root of all Zenject IOC containers. It contains dependencies which can be used across the entire application (e.g. EventDispatcher), if added as a parent contract name on a given Scene Context component. When the application is launched the Bootstrapper.cs script loads the scn_CUI scene.

    Relevant GameObjects:

    SceneContext: Installs dependencies (see Zenject for more information on how IOC works).

    This scene can be used for testing certain functionalities. To do this, add a SceneContext to your test scene (Right click in Hierarchy -> Zenject -> SceneContext). Under Parent Contract Names on the Scene Context script, add "Bootstrapper". When entering play mode, the Bootstrapper scene will automatically be loaded (Editor only feature), adding all dependencies from scn_Bootstrapper. See ZenjectDefaultSceneContractConfig for more information on default scene parents.

    scn_CUI

    This scene contains all UI elements (chat, workflow steps, etc.) displayed on the screen in front of the user.
