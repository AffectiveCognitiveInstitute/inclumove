    public uint GetNewRequestId()

    System.UInt32

    The new unsigned int request id.
    Remarks

    The first 3 bits are reserved for preventing duplicate ids between different system components. This makes it possible to use up to 8 different system components. The max amount of unique identifiers per system is 536870912. The value will reset if reaching the max value. The amount should prevent having duplicate values in the system anywhere.
